Summer Internship 2017

INDAGO

Indago is a web application that guides people into the tech industry by giving them the resources to learn the skills they need. We have three areas of knowledge that users can explore and get to know better: UI/UX Design, Ruby on Rails (Frontend) and Android.

01 /
Let's learn new stuff

Design Sprint

During this week our room walls were filled with post-its, stakeholders maps, value maps and positioning matrices. To validate any assumption our interns have conducted interviews and surveys, while developing Storyboard Sketching and working on a low-fi prototype.

02 /
The good stuff

Welcome party

After all the crazy 8’s, interviews and wireframes our interns were rewarded with a welcoming party! They were officially introduced to the whole Nest Collective crew, played Fernando Pessoa vs Gustavo Santos game, while eating “chouriça assada” and pizza!

Work, work, work

Development

Started with the application architecture definition, a server-client architecture, with a standalone RESTful API and an independent client. The API was developed using Ruby On Rails and the frontend using React with Redux and React Router. During the development one of the main requirements followed was to test the code, controllers and models.
